O que é Frontend?

É a camada visual e interativa de um site ou sistema.

O que é Cliente X Servidor?

O cliente é o que solicita o serviço, enquanto o servidor é o sistema se precessa a solicitação e entrega o recurso desejado.

Qual é o papel de HTML/CSS/JavaScript?

HTML estrutura o conteúdo, CSS define a aparência e JavaScript adiciona interação.

O que é W3C?

É o orgão responsavel por elaborar as atualizações, bem como documentar a evolução do HTML.
